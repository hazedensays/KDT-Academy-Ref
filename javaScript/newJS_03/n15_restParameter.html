** 나머지 매개변수(Rest Parameter) **

=> JS function 의 특징
- 함수 인자와 전달 인수의 갯수가 달라도 오류 발생 되지 않음
    ( 순차적으로 할당되고 나머지는 할당 되지 않음 )
- 함수 호출시 내부에 arguments 객체가 생성되며 배열과 같은 형태로 전달된 인자를 원소로 저장함.
     그러므로 가변인자를 사용하는 경우 arguments 객체를 활용함.
  
=> 나머지 매개변수(Rest Parameter)
   가변인자를 사용하는 경우 arguments 객체를 사용하지 않아도 됨.

=> 기본 매개변수(Default Parameter)  
     초기값 설정이 가능하고 인수 선언시 먼저 선언된 인수의 값을 활용할 수 있음
