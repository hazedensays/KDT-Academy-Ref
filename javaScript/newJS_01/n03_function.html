Default Parameters

=> 함수 정의 시에 Default Parameters를 정의하면,
   매개변수(parameter)없이 함수 호출 시에
   Default Parameters의 값이 적용된다.

😎 콘솔에서 확인하세용 😎

** 2. 화살표 함수(Arrow Function)

=> 제한사항   
- this나 super에 대한 바인딩이 없다.
- methods(객체의 속성) 로 사용될 수 없음.
- 생성자(Constructor) 함수로 사용할 수 없음.

=> 예시
 // 매개변수가 없는 경우 : 괄호 생략 불가능
 let foo = () => console.log('bar');
 // let foo = function() { console.log('bar'); }
 foo(); // bar


// 매개변수가 하나인 경우 : 괄호 생략 가능
let foo = x => x;
foo('bar'); // bar


// 매개변수가 여러 개인 경우 : 괄호 생략 불가능,
                             함수 body의 실행문이 1 문장이면 {} 생략 가능

let foo = (a, b) => a + b; // 간단하게 한줄로 표현할 땐 "{}", return 없이 값이 반환 가능
foo(1, 2); // 3

let foo = (a, b) => { return a + b }; //{} 사용 시에 return 생략 불가능
foo(1, 2); // 3

let foo = (a, b) => { a + b }; // "{}"를 사용했는데 return이 없을 때
foo(1, 2); // undefined
        
let foo = (a, b) => { // 여러줄 썼을 때
   let c = 3;
   return a + b + c;
}
foo(1, 2, 3) // 6
/*
간단하게 한줄로 표현할 땐 "{}", return 없이 값이 반환됩니다.
"{}"를 사용하면 값을 반환할 때 return을 사용해야합니다.
"{}"를 사용하고 return이 없으면 undefied를 반환합니다.
"{}"을 사용할 때는 여러줄을 썼을 때 사용합니다.
*/
        
// 객체를 반환할 때
let foo = () => ( { a: 1, b: 2, c: 3 } );
foo(); // { a: 1, b: 2, c: 3 };
콜백 함수에서도 사용할 수 있습니다.
